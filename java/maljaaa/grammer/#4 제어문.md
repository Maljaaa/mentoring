### for each
* J2SE 5.0(Java Platform, Standard Edition, java 개발/실행 환경)부터 추가됨

```
for(type var: iterate) {
  body-of-loop
}
```

* iterate : 루프를 돌릴 객체 -> 루프를 돌릴 수 있는 형태인 **배열 or ArrayList** 등
* iterate 객체에서 한 개씩 순차적으로 var에 대입되어 for문 수행
